# Scott Park Portfolio

A performance-first portfolio built with [Astro](https://astro.build/) and Tailwind CSS.

## Features
- **Performance-First**: Optimized for speed and efficiency.
- **Responsive Design**: Looks great on all devices.
- **Customizable**: Easily modify styles and content.

## Installation
1. Clone the repository: `git clone https://github.com/scottparkk/scottparkk.github.io`
2. Navigate to the project directory: `cd scottparkk.github.io`
3. Install dependencies: `npm install`

## Usage
- To start the development server, run: `npm run dev`
- For production build, run: `npm run build`
